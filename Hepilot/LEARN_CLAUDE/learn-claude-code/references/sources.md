# Claude Code 学习源

优先级从高到低如下。

## 1. 本地源码

- 路径：`../Claude-Code/`
- 用途：回答 Claude Code 当前实现、代码结构、调用链、控制流、数据结构等问题时优先使用。
- 说明：先查源码，再引用外部材料做补充。

## 2. shareAI-lab/learn-claude-code

- 地址：https://github.com/shareAI-lab/learn-claude-code/
- 用途：适合看学习路径、资料整理、实践经验、概念导航。
- 注意：这是学习资料仓库，不等于 Claude Code 官方源码实现。

## 3. Claude Code Source Analysis Handbook

- 地址：https://claude.whaty.org/posts/Claude-code-source-analysis-handbook.html
- 用途：适合做源码导读、模块理解、阅读顺序参考。
- 注意：把它当成辅助分析材料，不要用它覆盖本地源码事实。

## 使用原则

- 回答“Claude Code 现在实际怎么实现”时，先以本地源码为准。
- 回答“怎么开始学 Claude Code”时，可以更多使用 2 和 3。
- 当外部资料和本地源码表述不一致时，以本地源码为准，并明确指出差异。
