# 206. 反转链表例子推演过程

用例：

```text
1 -> 2 -> 3 -> 4 -> nil
```

初始：

- `prev = nil`
- `cur = 1`

## 第一步

先保存：

- `next = 2`

然后反转指针：

- `1.Next = nil`

再移动：

- `prev = 1`
- `cur = 2`

现在两部分状态是：

```text
已反转：1 -> nil
未处理：2 -> 3 -> 4 -> nil
```

## 第二步

保存：

- `next = 3`

反转：

- `2.Next = 1`

移动后：

- `prev = 2`
- `cur = 3`

状态变成：

```text
已反转：2 -> 1 -> nil
未处理：3 -> 4 -> nil
```

## 继续处理

同理继续处理 `3` 和 `4`。

最后得到：

```text
4 -> 3 -> 2 -> 1 -> nil
```

## 最终答案

循环结束时 `cur = nil`，`prev` 正好指向新的头结点，所以返回 `prev`。
