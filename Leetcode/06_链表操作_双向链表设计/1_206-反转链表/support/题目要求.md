# 206. 反转链表

给定单链表头结点，请反转链表并返回新头结点。

你需要在根目录 `answer.go` 中实现：

```go
type ListNode struct {
    Val  int
    Next *ListNode
}

func reverseList(head *ListNode) *ListNode
```

本地输入格式：

```text
n
val1 val2 ... valn
```

本地输出格式：

```text
长度 反转后的链表值1 反转后的链表值2 ...
```

空链表输出：

```text
0
```

只修改 `../answer.go`。

在 `support/` 目录下运行：

```bash
go test
```
