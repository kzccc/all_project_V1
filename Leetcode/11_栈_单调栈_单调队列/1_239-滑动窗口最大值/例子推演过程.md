# 239. 滑动窗口最大值 例子推演过程

我们用这个例子：

```text
nums = [1,3,-1,-3,5,3,6,7], k = 3
```

队列里存的是 **下标**，我下面同时写出对应值，方便看。

## 初始

```text
deque = []
ans = []
```

## i = 0, nums[i] = 1

- 队头没有过期元素
- 队尾没有比 1 小的元素
- 放入下标 0

```text
deque = [0]        // 值 [1]
```

窗口还没形成，不记录答案。

## i = 1, nums[i] = 3

- 队头没有过期元素
- 队尾下标 0 对应值 1，小于等于 3，弹出
- 放入下标 1

```text
deque = [1]        // 值 [3]
```

窗口还没形成，不记录答案。

## i = 2, nums[i] = -1

- 队头没有过期元素
- 队尾值 3 不小于 -1，不弹
- 放入下标 2

```text
deque = [1,2]      // 值 [3,-1]
```

第一个完整窗口 `[1,3,-1]` 形成，队头是 1，对应值 3：

```text
ans = [3]
```

## i = 3, nums[i] = -3

- 队头下标 1 还在窗口 `[1,3]` 内，不过期
- 队尾值 -1 不小于 -3，不弹
- 放入下标 3

```text
deque = [1,2,3]    // 值 [3,-1,-3]
ans = [3,3]
```

## i = 4, nums[i] = 5

- 队头下标 1 已经过期，因为 `1 <= 4-3`
- 弹出队头 1
- 然后从队尾开始清理：`-3 <= 5`，弹；`-1 <= 5`，弹
- 放入下标 4

```text
deque = [4]        // 值 [5]
ans = [3,3,5]
```

这里最能看出单调队列的价值：  
`5` 一进来，前面那些更小的数就彻底失去竞争力了。

## i = 5, nums[i] = 3

- 队头下标 4 没过期
- 队尾值 5 不小于 3，不弹
- 放入下标 5

```text
deque = [4,5]      // 值 [5,3]
ans = [3,3,5,5]
```

## i = 6, nums[i] = 6

- 队头下标 4 没过期
- 从队尾开始弹：`3 <= 6`，弹；`5 <= 6`，弹
- 放入下标 6

```text
deque = [6]        // 值 [6]
ans = [3,3,5,5,6]
```

## i = 7, nums[i] = 7

- 队头下标 6 没过期
- 从队尾开始弹：`6 <= 7`，弹
- 放入下标 7

```text
deque = [7]        // 值 [7]
ans = [3,3,5,5,6,7]
```

## 总结

这个过程里，队列维护的不是“当前窗口所有元素”，而是：

- 还活着的候选者
- 并且按值从大到小排好

所以每次取最大值时，直接拿队头就行。
