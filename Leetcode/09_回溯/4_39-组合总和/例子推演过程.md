# 39. 组合总和例子推演过程

用例：

```text
candidates = [2, 3, 6, 7]
target = 7
```

排序后仍然是：`[2, 3, 6, 7]`

## 从 2 开始

路径：`[2]`，剩余 `5`

继续选 `2`：

- 路径：`[2, 2]`，剩余 `3`

继续选 `2`：

- 路径：`[2, 2, 2]`，剩余 `1`
- 后面所有数都大于 `1`，剪枝返回

改选 `3`：

- 路径：`[2, 2, 3]`，剩余 `0`
- 得到一个合法组合

## 从 3 开始

路径：`[3]`，剩余 `4`

继续选 `3`：

- 路径：`[3, 3]`，剩余 `1`
- 无法继续，返回

## 直接选 7

路径：`[7]`

剩余 `0`，得到另一个合法组合。

## 最终答案

```text
[2 2 3]
[7]
```

这题最关键的是：允许重复取当前数，所以递归时起点不能后移。
