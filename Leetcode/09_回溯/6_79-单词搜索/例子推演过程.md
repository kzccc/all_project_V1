# 79. 单词搜索例子推演过程

用例：

```text
board = [
  "ABCE",
  "SFCS",
  "ADEE"
]
word = "ABCCED"
```

## 从 A 开始

左上角 `(0,0)` 是 `A`，和单词首字符匹配。

路径开始：

- `A`

## 继续找 B

从 `(0,0)` 出发的相邻位置里，右边 `(0,1)` 是 `B`，匹配成功。

路径变成：

- `A -> B`

## 继续找 C

从 `(0,1)` 往右，`(0,2)` 是 `C`。

路径变成：

- `A -> B -> C`

## 再找 C

继续从 `(0,2)` 搜，往下 `(1,2)` 还是 `C`。

路径变成：

- `A -> B -> C -> C`

## 找 E

从 `(1,2)` 往下 `(2,2)` 是 `E`。

路径变成：

- `A -> B -> C -> C -> E`

## 找 D

从 `(2,2)` 往左 `(2,1)` 是 `D`。

路径完成：

- `A -> B -> C -> C -> E -> D`

成功匹配整个单词，返回 `true`。

这题的关键是：每次进入一个格子都要临时标记，离开时再恢复，不然路径之间会互相污染。
