# 22. 括号生成例子推演过程

用例：

```text
n = 3
```

初始：

- `path = ""`
- `open = 0`
- `close = 0`

## 先连续放左括号

可以走到：

```text
(((
```

此时 `open = 3`，不能再放左括号，只能回头补右括号：

```text
((()))
```

得到第一个答案。

## 回到中间状态继续分叉

例如从 `(()` 开始：

- `(()(` -> `(()())`
- `(())(` -> `(())()`

继续枚举还能得到：

- `()(())`
- `()()()`

## 最终答案

```text
((()))
(()())
(())()
()(())
()()()
```

这题的关键不是“枚举所有长度为 `2n` 的串”，而是“在生成过程中始终保证前缀合法”。
