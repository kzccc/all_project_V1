# 51. N 皇后例子推演过程

用例：

```text
n = 4
```

## 第 0 行

尝试把皇后放在第 0 列：

```text
Q...
....
....
....
```

## 第 1 行

- 第 0 列冲突，同列不能放
- 第 1 列冲突，对角线不能放
- 第 2 列可以放

得到：

```text
Q...
..Q.
....
....
```

继续向下，如果后续无解，就回退。

## 换一种开局

当第 0 行放在第 1 列时：

```text
.Q..
....
....
....
```

继续搜索，最终能得到一个合法解：

```text
.Q..
...Q
Q...
..Q.
```

再换其他分支，还能得到另一个解：

```text
..Q.
Q...
...Q
.Q..
```

## 最终答案

`n = 4` 一共 2 组解。

这题的关键是把“冲突判断”压缩成列和两类对角线状态，而不是每次都扫整个棋盘。
