# 51. N 皇后

## 题目概述

在 `n x n` 棋盘上放置 `n` 个皇后，要求任意两个皇后不能互相攻击，返回所有摆放方案。

## 面试最佳解法

首选 `回溯 + 列/对角线判冲突`。

这题的核心不是暴力摆棋盘，而是每一行只放一个皇后，并快速判断当前位置是否合法。

## 核心思路

从第 0 行开始，尝试把皇后放到每一列。

如果当前位置所在的：

- 列没有被占用
- 主对角线没有被占用
- 副对角线没有被占用

就可以放下皇后并继续递归下一行。

## 复杂度

属于典型指数级回溯问题，实际效率主要依赖剪枝。

## 容易写错的点

1. 对角线编号写错。
2. 回溯返回时忘记恢复列和对角线状态。
3. 棋盘恢复现场漏写。
