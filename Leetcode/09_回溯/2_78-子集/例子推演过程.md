# 78. 子集例子推演过程

用例：

```text
nums = [1, 2, 3]
```

初始：`path = []`

先把当前 `path` 记入答案，所以空集先进入答案。

## 从 1 开始选

- 选 `1`，得到 `[1]`
- 在 `[1]` 基础上继续选 `2`，得到 `[1, 2]`
- 在 `[1, 2]` 基础上继续选 `3`，得到 `[1, 2, 3]`
- 回退后尝试 `[1, 3]`

## 不选 1，改从 2 开始

- 得到 `[2]`
- 再扩展成 `[2, 3]`

## 最后只选 3

- 得到 `[3]`

最终顺序：

```text
[]
[1]
[1 2]
[1 2 3]
[1 3]
[2]
[2 3]
[3]
```

每个节点本身就是一个答案，这就是这题和全排列最大的区别。
