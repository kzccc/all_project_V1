# 17. 电话号码的字母组合例子推演过程

用例：

```text
digits = "23"
```

映射关系：

- `2 -> abc`
- `3 -> def`

## 第 1 层

先处理第 0 位 `2`：

- 选 `a`
- 选 `b`
- 选 `c`

## 以 `a` 开头继续

当前路径：`a`

处理第 1 位 `3`：

- 加 `d`，得到 `ad`
- 加 `e`，得到 `ae`
- 加 `f`，得到 `af`

## 以 `b` 开头继续

得到：

- `bd`
- `be`
- `bf`

## 以 `c` 开头继续

得到：

- `cd`
- `ce`
- `cf`

## 最终答案

```text
ad
ae
af
bd
be
bf
cd
ce
cf
```

每一层负责一位数字，每条根到叶子的路径就是一个完整字符串。
