# 131. 分割回文串例子推演过程

用例：

```text
s = "aab"
```

## 从第 0 位开始切

先试最短回文串 `a`：

- 路径：`[a]`

继续从第 1 位开始切。

## 从第 1 位开始切

先试 `a`：

- 路径：`[a, a]`

再从第 2 位开始切，只能取 `b`：

- 路径：`[a, a, b]`
- 到达末尾，得到一个答案

## 回退后换另一种切法

回到第 0 位时，尝试更长的回文串 `aa`：

- 路径：`[aa]`

剩余部分是 `b`：

- 路径：`[aa, b]`
- 得到第二个答案

## 最终答案

```text
[a a b]
[aa b]
```

这题的关键在于：不是所有切法都递归，而是只有当前子串本身是回文时才继续向下搜索。
