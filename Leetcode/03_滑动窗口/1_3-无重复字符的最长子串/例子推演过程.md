# 3. 无重复字符的最长子串例子推演过程

用例：

```text
s = "pwwkew"
```

初始：

- `left = 0`
- `best = 0`
- `lastIndex = {}`

## 读到 `p`

- 没出现过
- 当前窗口是 `"p"`
- 长度 `1`

更新：

```text
best = 1
lastIndex['p'] = 0
```

## 读到第一个 `w`

- 没出现过
- 当前窗口是 `"pw"`
- 长度 `2`

更新：

```text
best = 2
lastIndex['w'] = 1
```

## 读到第二个 `w`

- `w` 上次出现在位置 `1`
- 它还在当前窗口里
- 所以把 `left` 跳到 `2`

现在窗口变成：

```text
"w"
```

## 继续读到 `k`、`e`

窗口逐步变成：

```text
"wk"
"wke"
```

长度更新到 `3`。

## 最后读到 `w`

- `w` 上次出现在位置 `2`
- 仍在当前窗口内
- 把 `left` 移到 `3`

窗口变成：

```text
"kew"
```

长度也是 `3`。

## 最终答案

最长无重复子串长度是：

```text
3
```
