# LeetCode Hot 100 面试最佳解法分类题单

整理时间：2026-04-30  
官方题单来源：<https://leetcode.cn/studyplan/top-100-liked/>

## 说明

这份题单只做一件事：把当前官方 `LeetCode 热题 100` 按“面试里最主流、最稳、最好讲清楚的最佳解法”归类。

判类规则：

1. 一道题只归入一个主类，不做重复归类。
2. 如果有多种可行解，以当前面试里更常写、更容易讲清楚、更容易一次写对的那种解法为准。
3. 分类优先看“主解法技术路线”，而不是题目长得像什么数据结构。

几个容易分歧的例子：

- `239. 滑动窗口最大值` 归 `栈 / 单调结构`，主解法是 `单调队列`，不归普通滑动窗口。
- `437. 路径总和 III` 归 `前缀和`，主解法是 `树上前缀和 + 哈希`，不归普通二叉树遍历。
- `300. 最长递增子序列` 归 `贪心 + 二分`，不归朴素 DP。
- `5. 最长回文子串` 归 `双指针 / 中心扩展`，不归 DP。
- `23. 合并 K 个升序链表` 归 `堆 / 优先队列`，不归普通链表题。
- `138. 随机链表的复制` 归 `哈希表 / 映射`，主解法按 `哈希映射` 归类。
- `287. 寻找重复数` 归 `双指针 / 快慢指针`，主解法按 `Floyd 判环` 归类。
