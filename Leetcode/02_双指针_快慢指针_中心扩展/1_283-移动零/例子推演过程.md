# 283. 移动零例子推演过程

下面用一个中等复杂度例子完整推演：

```text
nums = [0, 1, 0, 3, 12, 0, 5]
```

初始状态：

- `slow = 0`
- `fast = 0`

目标：

- 把所有非零元素按原顺序挪到前面
- 把所有零移到后面

## 第 1 步

`fast = 0`，当前值是 `0`

- 这是零，不处理交换
- `slow` 不动

数组状态：

```text
[0, 1, 0, 3, 12, 0, 5]
```

指针状态：

- `slow = 0`
- `fast = 1`

## 第 2 步

`fast = 1`，当前值是 `1`

- 这是非零
- 交换 `nums[slow]` 和 `nums[fast]`
- 也就是交换 `nums[0]` 和 `nums[1]`

交换后：

```text
[1, 0, 0, 3, 12, 0, 5]
```

然后：

- `slow++`

指针状态：

- `slow = 1`
- `fast = 2`

此时含义是：

- 下标 `0` 位置已经放好了第一个非零元素 `1`

## 第 3 步

`fast = 2`，当前值是 `0`

- 跳过
- `slow` 不动

数组状态：

```text
[1, 0, 0, 3, 12, 0, 5]
```

指针状态：

- `slow = 1`
- `fast = 3`

## 第 4 步

`fast = 3`，当前值是 `3`

- 这是非零
- 交换 `nums[1]` 和 `nums[3]`

交换后：

```text
[1, 3, 0, 0, 12, 0, 5]
```

然后：

- `slow = 2`
- `fast = 4`

此时前面已经排好：

```text
[1, 3]
```

## 第 5 步

`fast = 4`，当前值是 `12`

- 这是非零
- 交换 `nums[2]` 和 `nums[4]`

交换后：

```text
[1, 3, 12, 0, 0, 0, 5]
```

然后：

- `slow = 3`
- `fast = 5`

## 第 6 步

`fast = 5`，当前值是 `0`

- 跳过

数组状态：

```text
[1, 3, 12, 0, 0, 0, 5]
```

指针状态：

- `slow = 3`
- `fast = 6`

## 第 7 步

`fast = 6`，当前值是 `5`

- 这是非零
- 交换 `nums[3]` 和 `nums[6]`

交换后：

```text
[1, 3, 12, 5, 0, 0, 0]
```

然后：

- `slow = 4`
- `fast = 7`

遍历结束。

## 最终结果

```text
[1, 3, 12, 5, 0, 0, 0]
```

## 这个过程说明了什么

整个过程中：

1. `fast` 负责找下一个非零元素。
2. `slow` 负责维护“下一个该放非零元素的位置”。
3. 每找到一个非零元素，就把它交换到前面。
4. 因为 `fast` 是从左到右扫描，所以非零元素进入前半部分的顺序不会乱。

这就是这道题双指针解法最核心的面试表达方式。
