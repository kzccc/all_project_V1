# 283. 移动零

## 训练目标

给定一个整数数组 `nums`，把所有 `0` 移动到数组末尾，同时保持非零元素的相对顺序不变。

你需要在根目录的 `answer.go` 中实现：

```go
func moveZeroes(nums []int)
```

要求：

- 原地修改数组
- 不返回新数组
- 优先使用面试里最主流的双指针写法

## 本地训练输入格式

本地 runner 使用下面的输入格式：

```text
n
nums[0] nums[1] ... nums[n-1]
```

也可以写成一行，按空白分隔读取：

```text
n nums[0] nums[1] ... nums[n-1]
```

例如：

```text
5
0 1 0 3 12
```

## 本地训练输出格式

输出移动零之后的整个数组，元素之间用空格分隔。

例如：

```text
1 3 12 0 0
```

## 你应该修改哪个文件

只改：
