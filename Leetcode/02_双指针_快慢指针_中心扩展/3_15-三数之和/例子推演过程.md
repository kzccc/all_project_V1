# 15. 三数之和例子推演过程

用例：

```text
nums = [-1, 0, 1, 2, -1, -4]
```

先排序：

```text
[-4, -1, -1, 0, 1, 2]
```

## 固定 `i = 0`

固定 `-4`，目标变成找两个数和为 `4`。

- `left = 1`，`right = 5`
- `-1 + 2 = 1`，太小，左指针右移
- 继续尝试后仍然找不到

所以以 `-4` 开头没有答案。

## 固定 `i = 1`

固定 `-1`，目标变成找两个数和为 `1`。

- `left = 2`，`right = 5`
- `-1 + 2 = 1`

得到第一个答案：

```text
[-1, -1, 2]
```

然后左右指针同时移动，并跳过重复值。

接着：

- `left = 3`，`right = 4`
- `0 + 1 = 1`

得到第二个答案：

```text
[-1, 0, 1]
```

## 固定 `i = 2`

`nums[2]` 也是 `-1`，和前一个固定值重复，直接跳过。

## 最终答案

```text
[-1, -1, 2]
[-1, 0, 1]
```

关键点就是：排序后用双指针找两数之和，并在三个位置都认真去重。
