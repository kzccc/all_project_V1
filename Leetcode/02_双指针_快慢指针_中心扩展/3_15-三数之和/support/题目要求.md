# 15. 三数之和

给定一个整数数组，返回所有和为 `0` 且不重复的三元组。

你需要在根目录 `answer.go` 中实现：

```go
func threeSum(nums []int) [][]int
```

本地输入格式：

```text
n
nums[0] nums[1] ... nums[n-1]
```

本地输出格式：

- 第一行输出三元组数量
- 之后每行格式为：`3 a b c`

说明：

- `runner` 会把每个三元组和整体答案都排序后输出，避免多解顺序影响判题。

只修改 `../answer.go`。

在 `support/` 目录下运行：

```bash
go test
```
