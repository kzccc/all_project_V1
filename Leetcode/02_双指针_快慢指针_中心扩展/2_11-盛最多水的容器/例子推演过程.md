# 11. 盛最多水的容器例子推演过程

用例：

```text
height = [1, 8, 6, 2, 5, 4, 8, 3, 7]
```

初始：

- `left = 0`，高度 `1`
- `right = 8`，高度 `7`

面积：

```text
min(1, 7) * 8 = 8
```

因为左边更短，所以移动 `left`。

## 第二步

- `left = 1`，高度 `8`
- `right = 8`，高度 `7`

面积：

```text
min(8, 7) * 7 = 49
```

更新答案为 `49`。

因为右边更短，所以移动 `right`。

## 第三步

- `left = 1`，高度 `8`
- `right = 7`，高度 `3`

面积：

```text
min(8, 3) * 6 = 18
```

没有超过 `49`，继续移动较短边 `right`。

## 后续过程

继续按同样规则收缩，虽然还会遇到很多组合，但都不会超过 `49`。

## 最终答案

最大面积来自：

- 左边高度 `8`
- 右边高度 `7`
- 宽度 `7`

所以答案是：

```text
49
```
