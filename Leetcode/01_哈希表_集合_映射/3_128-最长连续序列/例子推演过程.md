# 128. 最长连续序列例子推演过程

用例：

```text
nums = [100, 4, 200, 1, 3, 2]
```

先放入集合：

```text
{100, 4, 200, 1, 3, 2}
```

## 枚举 `100`

- `99` 不在集合里，说明 `100` 是起点
- 向右看 `101` 不在集合里
- 当前连续长度是 `1`

## 枚举 `4`

- `3` 在集合里
- 说明 `4` 不是起点，跳过

## 枚举 `200`

- `199` 不在集合里，是起点
- `201` 不在集合里
- 当前连续长度还是 `1`

## 枚举 `1`

- `0` 不在集合里，是起点
- 继续向右扩展：
  `2` 在，`3` 在，`4` 在，`5` 不在
- 所以这一段长度是 `4`

## 最终答案

最长连续序列是：

```text
1, 2, 3, 4
```

长度为：

```text
4
```

关键就在于：只有起点才扩展，避免重复计算。
