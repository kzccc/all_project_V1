# 1. 两数之和

给定一个整数数组 `nums` 和目标值 `target`，请返回两个下标，使得 `nums[i] + nums[j] = target`。

你需要在根目录 `answer.go` 中实现：

```go
func twoSum(nums []int, target int) []int
```

本地输入格式：

```text
n
nums[0] nums[1] ... nums[n-1]
target
```

本地输出格式：

```text
i j
```

说明：

- 题目保证恰好存在一组答案。
- 测试数据按唯一答案准备。

只修改 `../answer.go`。

