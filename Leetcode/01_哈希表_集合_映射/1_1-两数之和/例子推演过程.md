# 1. 两数之和例子推演过程

用例：

```text
nums = [2, 7, 11, 15], target = 9
```

初始状态：

- 哈希表 `indexByValue = {}`

## 第 1 步

看 `nums[0] = 2`。

- 需要找的配对值是 `9 - 2 = 7`
- 哈希表里没有 `7`
- 把 `2 -> 0` 放进去

此时：

```text
indexByValue = {2: 0}
```

## 第 2 步

看 `nums[1] = 7`。

- 需要找的配对值是 `9 - 7 = 2`
- 哈希表里有 `2`，对应下标 `0`

所以答案就是：

```text
[0, 1]
```

## 为什么这样做一定对

因为当我们扫描到当前位置时，哈希表里只存了左边已经出现过的数。

如果 `target - nums[i]` 已经在表里，就说明找到了一个更早的下标 `j`，满足：

```text
nums[j] + nums[i] = target
```

而且每个元素最多被用一次。
