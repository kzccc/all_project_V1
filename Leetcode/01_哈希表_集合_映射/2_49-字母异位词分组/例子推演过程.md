# 49. 字母异位词分组例子推演过程

用例：

```text
["eat", "tea", "tan", "ate", "nat", "bat"]
```

初始状态：

- 哈希表 `groups = {}`

## 处理 `"eat"`

- 排序后得到 `"aet"`
- 放入 `groups["aet"]`

```text
"aet" -> ["eat"]
```

## 处理 `"tea"`

- 排序后还是 `"aet"`
- 放入同一组

```text
"aet" -> ["eat", "tea"]
```

## 处理 `"tan"`

- 排序后得到 `"ant"`

```text
"aet" -> ["eat", "tea"]
"ant" -> ["tan"]
```

## 继续处理 `"ate"`、`"nat"`、`"bat"`

最终得到：

```text
"aet" -> ["eat", "tea", "ate"]
"ant" -> ["tan", "nat"]
"abt" -> ["bat"]
```

## 最终答案

把哈希表里的所有分组取出来即可。

本质上，这题就是把“是否互为异位词”转换成“排序后的 key 是否相同”。
