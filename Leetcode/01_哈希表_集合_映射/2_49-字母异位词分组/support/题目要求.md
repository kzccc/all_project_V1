# 49. 字母异位词分组

给定若干字符串，请把字母异位词分到同一组。

你需要在根目录 `answer.go` 中实现：

```go
func groupAnagrams(strs []string) [][]string
```

本地输入格式：

```text
n
str1
str2
...
strn
```

本地输出格式：

- 第一行输出分组数量
- 之后每行输出：`组内元素个数 字符串1 字符串2 ...`

说明：

- 为了便于本地判题，`runner` 会把每组字符串排序，再把所有组按字典序排序后输出。

只修改 `../answer.go`。

在 `support/` 目录下运行：

```bash
go test
```
